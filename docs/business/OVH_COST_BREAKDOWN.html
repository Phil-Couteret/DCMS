📋 Detailed Service Breakdown

1. VPS (Virtual Private Server)

VPS-2 (Recommended for Starting):

  • RAM: 12GB
  • CPU: 6 vCores
  • Storage: 100GB NVMe SSD
  • Bandwidth: 250 Mbps guaranteed, unlimited traffic
  • Backups: Automatic daily backups included
  • Anti-DDoS: Included
  • Monthly: €6.75
  • Yearly: €81.00

Capacity: Handles 5,000-15,000 bookings/month comfortably

VPS-3 (For Higher Traffic):

  • RAM: 16-24GB (verify current spec on OVH)
  • CPU: 8 vCores
  • Storage: 150-200GB NVMe SSD (verify current spec)
  • Bandwidth: 1.5-2 Gbps guaranteed, unlimited traffic
  • Backups: Automatic daily backups included
  • Anti-DDoS: Included
  • Monthly: €11.25-12.75 (verify current pricing)
  • Yearly: €135-153

Capacity: Handles 15,000-30,000 bookings/month comfortably

Note: VPS-3 specifications may vary - check OVH website for exact current configuration

2. Managed PostgreSQL Database

Starter Configuration (VPS-2 setup):

  • RAM: 2GB dedicated
  • CPU: 1 vCPU dedicated
  • Storage: 100GB SSD
  • Backups: Automated daily backups (30 days retention)
  • High Availability: Option available (extra cost)
  • Monitoring: Included
  • Monthly: €20.00
  • Yearly: €240.00

Enhanced Configuration (VPS-3 setup):

  • RAM: 4GB dedicated
  • CPU: 2 vCPU dedicated
  • Storage: 100GB SSD
  • Backups: Automated daily backups (30 days retention)
  • High Availability: Option available (extra cost)
  • Monitoring: Included
  • Monthly: €40.00
  • Yearly: €480.00

Note: Database storage can be expanded if needed (extra cost)

3. Object Storage (Files & Backups)

  • Service: OVH Object Storage
  • Storage: 250GB
  • Purpose: User uploads, documents, photos, application files
  • Bandwidth: Unlimited (within OVH network)
  • Monthly: €5.00
  • Yearly: €60.00

Note: Can be expanded if needed (€0.02/GB/month additional)

4. Business Email

  • Service: OVH Business Email (Exchange or Mail)
  • Mailboxes: 5 mailboxes (info@, booking@, admin@, support@, contact@)
  • Storage per mailbox: 10GB (50GB total)
  • Features: Webmail, IMAP/POP3, calendar, contacts
  • Monthly: €2/month per mailbox = €10.00 total
  • Yearly: €120.00

Alternative: Can reduce to 3 mailboxes = €6/month if budget is tight

5. Domain Registration

  • Service: OVH Domain Registration
  • Domain: .com or .es domain
  • Registration: €12-20/year (paid upfront)
  • Privacy: WHOIS privacy included (GDPR)
  • DNS Management: Included
  • Monthly equivalent: €1.50
  • Yearly: €18.00

Note: Typically paid yearly upfront, not monthly

6. SSL Certificate

  • Service: Let's Encrypt (free)
  • Cost: €0.00
  • Auto-renewal: Yes
  • Coverage: Single domain or wildcard

7. Backups

  • VPS Backups: Included with VPS (daily automatic backups)
  • Database Backups: Included with Managed PostgreSQL (30 days retention)
  • Additional Backup Storage: Can add Object Storage for off-site backups
  • Cost: €0.00 (included)

8. DNS Management

  • Service: OVH DNS
  • Included with: Domain registration
  • Cost: €0.00
